
# Dunder Mifflin project

A webshop created to be able to buy paper and such.



## Authors

## Deployment

To deploy this project frontend + backend

```bash
  cd client
```

```bash
  npm run dev
```

```bash
  cd server
```
```bash
  cd api
```
```bash
  dotnet run
```
